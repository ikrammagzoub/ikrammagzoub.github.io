1.3: Image Straightening

In this portion of the project, I created a method that automatically finds the best angle of rotation to straighten an image. To achieve that, I followed these steps:

  1. For an image, sample rotation angles from −10° to 10°, in 1° increments.
  2. For each angle, rotate the image by an angle between -5 to 5.
  3. Crop the image so we only have the center.
  4. Compute the edges of the images in the same way we did in part 1.2.
  5. Compute the gradient angles of the image using arctan2.
  6. Count the number of horizontal and vertical edges that are within one degree of abs(90, 180).
  7. Choose the rotation with the highest sum of vertical and horizontal edges.
